== README

# Descripton
FitTable is a Ruby on Rails project that allow people to create table routines for gym. Users can create his own exercises and share with
the community to allow other users to add it to his personal tables.

## Setup

1. $ sudo yum install ImageMagick
2. Clone the repo # $ git clone https://github.com/ayoseturru/fittable
3. $ cd fittable
4. $ bundle 
5. $ rake db:setup
6. $ rails s
